# BuildTrack Frontend 🏗️

React frontend for the BuildTrack construction site tracking app.

## Live Demo

https://buildtrack-frontend-ten.vercel.app

## Tech Stack

- React + Vite
- React Router DOM
- Axios
- Tailwind CSS

## Backend

https://github.com/kuzmavasil74/buildtrack

## Local Development

```bash
npm install
npm run dev
```

## Environment Variables

```
VITE_API_URL=https://buildtrack-api-x1xl.onrender.com
```
